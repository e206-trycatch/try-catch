# 기능 명세서

| 대분류 | 기능명 | 설명 |
| --- | --- | --- |
| 로그인 | 로그인 실패 | 인증 실패 시 오류 메시지를 표시한다. |
| 로그인 | 아이디/비밀번호 로그인 | 아이디/비밀번호 입력 후 인증을 수행하고 로그인한다. |
| 계정 복구 | 로그인/비밀번호 찾기 안내 | 계정 분실 시 복구 방법(이메일/SMS 등)을 안내한다. |
| 회원가입 | 가입 실패 처리 | 입력 오류/중복 등 실패 사유를 안내한다. |
| 회원가입 | 신규 사용자 등록 | 입력값 검증 및 중복 확인 후 회원가입을 처리한다. |
| 회원가입 | 회원가입 정보 입력 | 아이디/비밀번호/닉네임 등 필수 정보를 입력한다. |
| 회원 정보 | 회원 정보 수정 | 닉네임/프로필 정보를 수정한다. |
| 랭킹 | 멀티 모드 랭킹 조회 | 멀티 모드 플레이 결과를 기반으로 팀 랭킹을 조회한다. |
| 기록 | 제출 코드 조회 | 본인의 제출 코드 히스토리를 조회한다. |
| 랭킹 | 싱글 모드 랭킹 조회 | 싱글 모드 플레이 결과를 기반으로 사용자 랭킹을 조회한다. |
| 모드 선택 | 싱글/멀티 모드 선택 | 사용자가 싱글 또는 멀티 모드를 선택할 수 있다. |
| 테마 진행도 | 테마 진행도 | 테마 완료 여부를 도장(스탬프) 형태로 표시한다. |
| 방 생성 | 테마 선택하기 | 사용자는 테마 목록에서 원하는 테마를 선택한 후 방을 생성한다. |
| 게임 설정 | 게임 설정 선택 | 사용자는 포지션/언어프레임워크를 선택한다. |
| 방 입장 | 초대 코드 입력 | 사용자는 공유받은 초대 코드를 입력하여 멀티 방에 참여할 수 있다. |
| 테마 탐색 | 테마 상세 보기 | 테마 설명 및 난이도를 확인할 수 있다. |
| 게임 설정 | 게임 설정 선택 | 방 이름/테마/언어프레임워크를 선택 후 방을 생성한다. |
| 방 정보 | 방 정보 조회 | 방 이름/테마/난이도/설정값을 제공한다. |
| 방 코드 | 방 코드 복사 | 방장이 방 코드를 복사해서 다른 사용자를 초대한다. |
| 게임 시작 | 게임 시작 | 방장이 게임 시작을 수행한다. |
| 참가자 관리 | 참가/퇴장 처리 | 입장/퇴장 시 참가자 상태를 동기화한다. |
| 준비 상태 | Ready 상태 변경 | 참가자가 준비 상태를 변경하고 상태를 공유한다. |
| 참가자 관리 | 참가자 목록 표시 | 방 참가자 리스트를 표시한다. |
| [멀티] 준비 | 준비 처리 | 모든 사용자가 준비 완료 시 문제 풀이를 시작한다. |
| [싱글] 문제 | 문제 선택 | 사용자는 문제를 선택할 수 있다. |
| [멀티] 문제 | 문제 선택 | 멀티 모드에서 문제를 선택할 수 있다. |
| 디버깅 정보 | 디버깅 정보 탐색 | 탭 전환/스크롤 등으로 디버깅 정보를 확인한다. |
| 문제 진행도 | 문제 완료 상태 갱신 | 문제 해결 시 완료 상태를 갱신하고 도장을 활성화한다. |
| 디버깅 정보 | 에러 상황 설명 제공 | 에러 발생 조건과 상황을 텍스트로 제공한다. |
| 제출/채점 | 제출 | 코드 제출을 수행한다. |
| 제출/채점 | 제출 결과 처리 | 제출 후 성공/실패 여부를 판정하고 결과를 처리한다. |
| 음성 | 음성 유지 | 게임 시작 후에도 WebRTC 음성을 유지한다. |
| 채팅 | 게임 중 채팅 | 게임 진행 중 팀 채팅을 지원한다. |
| 코드 에디터 | 언어/프레임워크 전환 | 멀티 프레임워크 코드 파일을 전환할 수 있다. |
| 실시간 UI | 타이머 동기화 | 남은 시간을 서버 기준으로 동기화하여 표시한다. |
| 힌트 | 힌트 제공 | 힌트 1/2/3 등을 제공한다. |
| 코드 에디터 | 코드 에디터 제공 | Monaco Editor 기반 코드 편집 기능을 제공한다. |
| 디버깅 정보 | 에러 로그 표시 | 문제에 포함된 에러 로그를 표시한다. |
| 게임 상태 | 중도 포기 | 게임 중 나가기/포기 기능을 제공한다. |
| 결과 표시 | 실패 원인 표시 | 에러 로그 등 실패 원인을 제공한다. |
| 결과 표시 | 성공 결과 표시 | 문제/난이도/시간/총점 등 성공 결과를 표시한다. |
| 결과 표시 | 실패 결과 표시 | 문제/난이도/시간/남은 목숨 등 실패 결과를 표시한다. |
| 로딩 | 개발 TMI 표시 | 로딩 중 개발 관련 지식/문구를 표시한다. |
| 재시작 | 다시하기 | 동일 문제로 다시 플레이한다. |
| 로딩 | 채점 진행률 표시 | AI/채점 진행 상태를 표시한다. |
