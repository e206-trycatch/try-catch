# 아이디어기획_도출_내용

# 프로젝트 아이디어 기획 요약

## 📌 개요

SSAFY WebRTC 기반 프로젝트를 위한 팀 아이디어 브레인스토밍 결과물입니다.

---

## 최종 아이디어 후보

### 1. 🕵️ 디지털 에러 탈출 (실시간 멀티플레이 디버깅 게임)

**컨셉:** 디버깅·로그 분석 과정을 방탈출/타임어택 멀티플레이 게임으로 변환

**타겟:** SSAFY 교육생, 개발자 지망생, IT 동아리

**핵심 기능:**

- 협력 퍼즐: 팀원별로 다른 단서를 조합해야 해결 가능
- 가상 터미널: 웹에서 리눅스 명령어로 단서 탐색
- 타임 어택: 제한 시간 내 서버 복구 미션
- 실시간 동기화: WebSocket 기반 팀원 상태 공유

**게임 흐름:**

1. 고립 단계 (0~5분): 본인 파일만 열람, 채팅으로 정보 공유
2. 협력 단계 (5~10분): 모든 파일 접근 가능, 실시간 동시 편집

**기술 스택:**

- Frontend: React + Monaco Editor + WebSocket
- Backend: Spring Boot + Docker(샌드박스) + Redis
- 채점: 격리된 컨테이너에서 코드 실행

---

### 2. 👭 병실 친구 아바타 (소아병동 아이들을 위한 서비스)

**컨셉:** 병실에서 오래 지내는 아이가 아바타 친구와 함께 외로움을 줄이는 웹앱

**타겟:** 장기 입원 아동 + 보호자

**핵심 기능:**

- 아이-아바타 1:1 만남: AI 기반 대화 및 놀이
- 소그룹 모임: 봉사자와 3~4명 아이들이 화상으로 만남
- 아바타 키우기: 일상 활동(물 마시기, 스트레칭 등)으로 포인트 획득
- 얼굴 마스크 모드: 프라이버시 보호용 얼굴 필터
- 정서 케어: 기분 체크 및 난이도별 미션 추천

**기술 스택:**

- Frontend: React + WebRTC + MediaPipe(얼굴인식)
- Backend: Spring Boot + WebSocket(STOMP)
- AI: TTS + 비지움 기반 입모양 동기화
- Infra: TURN/STUN 서버 (병원 네트워크 대응)

**안전 설계:** 보호자 동반 필수, 실명/연락처 교환 금지, 녹화 차단

---

### 3. 🍳 자취 요리 앱 (AI 요리 어시스턴트)

**컨셉:** 자취생을 위한 식재료 관리 + AI 요리 도우미

**타겟:** 자취생, 1인 가구, 요리 초보

**핵심 기능:**

- 최저가 비교: 식재료 가격 비교 (다나와 식재료 버전)
- AI 요리 도우미: RAG 기반 레시피 학습, 실시간 요리 가이드
- 식재료 관리: 영수증 OCR, 유통기한 자동 설정
- 냉장고 촬영: 사진으로 보유 식재료 인식
- 보관 팁 제공: 오래 보관할 수 있는 방법 안내

## 🔧 공통 기술 요소

**WebRTC 관련:**

- P2P 영상/음성 통신
- WebSocket 시그널링
- TURN/STUN 서버 구성

**실시간 동기화:**

- Redis Pub/Sub
- STOMP 기반 메시징
- 동시 편집 (CRDT/파일 잠금)

**AI 활용:**

- 얼굴 인식: MediaPipe, TensorFlow.js
- 음성 분석: Whisper API, Web Audio API
- 자연어 처리: LLM 기반 피드백/힌트

**샌드박스 실행:**

- Docker 컨테이너 격리
- 타임아웃/리소스 제한
- 테스트 케이스 기반 채점
